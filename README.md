# Snowflake Self Service Reporting Portal

This Streamlit app lets users build dynamic reports from Snowflake tables with multiple filters, AND/OR logic, and CSV download functionality.

## Features
- Select reports from Snowflake-configured list
- Add multiple filters with operators: `=, >, <, >=, <=, LIKE, IN, IS NULL, IS NOT NULL`
- Choose `AND/OR` between filters
- Download query results as CSV
